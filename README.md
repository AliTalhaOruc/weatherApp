# Weather App ☁️

##  HavaDurumu (Expo)
Hava durumuna göre kombin öneren mobil uygulama.

##  Backend (Node.js)
OpenAI API ile iletişim sağlar.

## Güvenlik
API keyler güvenlik sebebiyle paylaşılmamıştır.

## Kurulum

### Mobile
cd mobile
npm install
npx expo start

### Backend
cd backend
npm install
node server.js
